Worttrennung
-a|ti|on/-ie|rung

Bedeutung

oftmals konkurrierende Endungen von Substantiven, die von Verben auf -ieren abgeleitet sind. Oft stehen beide Bildungen ohne Bedeutungsunterschied nebeneinander, z. B. Isolation/Isolierung, Konfrontation/Konfrontierung, doch zeichnen sich insofern Bedeutungsnuancen ab, als die Wörter auf -ation stärker das Ergebnis einer Handlung bezeichnen, während die Parallelbildung auf -ierung mehr das Geschehen oder die Handlung betont, wofür allerdings auch nicht selten die Bildung auf -ation gebraucht wird
